## Recommended Way To Use This Site

1. Start with [Program Overview](/start-here/program-overview/) to understand the complete roadmap.
2. Use the new [Learning Overview](/learning/overview/) when you want learner-facing concept pages, starting with the multi-page [JavaScript Overview](/learning/javascript/overview/).
3. Use [Training Format](/start-here/training-format/) to plan classroom rhythm, labs, and assessments.
4. Teach the [phase roadmap](/roadmap/phase-1-foundation-runtime/) in order, then use each module page during delivery.
5. Finish by aligning students to a [capstone project](/projects/capstone-projects/) and the [tools reference](/projects/tools-technologies/).
