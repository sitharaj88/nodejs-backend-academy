Professional backend work depends on tooling discipline, not only language knowledge.

This track turns Module 03 into a real learning path covering package management, project setup, scripts, configuration, linting, building, and test workflow.

## What You Will Learn

- npm basics, package metadata, and dependency discipline
- semantic versioning and lockfile behavior
- project setup, scripts, environment configuration, and startup conventions
- code-quality tools, type checking, formatting, builds, and test automation

## Recommended Order

1. [Package Management, Semver, and Lockfiles](/learning/tooling/package-management-semver-lockfiles/)
2. [Project Setup, Scripts, and Configuration](/learning/tooling/project-setup-scripts-configuration/)
3. [Code Quality, Builds, and Test Workflow](/learning/tooling/code-quality-builds-test-workflow/)
4. [Modern Tooling Coverage](/learning/tooling/modern-tooling-coverage/)

## Goal

By the end of this track, learners should be able to set up a clean Node.js project with sensible scripts, reproducible dependency management, and a maintainable quality workflow.
