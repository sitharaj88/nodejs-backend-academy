This page maps the brochure tooling list into teaching categories so students understand where each tool fits in the program.

## Languages and Runtime

- JavaScript ES6+
- Node.js runtime and built-in modules
- TypeScript awareness for backend projects

## Frameworks and API Layer

- Express.js
- REST API design
- Swagger or OpenAPI basics
- GraphQL fundamentals
- Socket.IO
- webhook integration patterns

## Databases and Caching

- MongoDB
- Mongoose
- PostgreSQL fundamentals
- MySQL fundamentals
- Redis concepts

## Security and Quality

- JWT
- bcrypt
- Helmet
- CORS
- rate limiting
- validation and sanitization
- Jest or Vitest
- Supertest
- logging and debugging workflows

## DevOps and Delivery

- Git and GitHub workflows
- GitHub Actions
- Docker
- PM2
- Nginx
- cloud or VPS deployment basics

## Where These Tools Appear In The Roadmap

| Phase | Main tool categories |
| --- | --- |
| Phase 1 | JavaScript, Node.js runtime, npm, linting, formatting, Git setup |
| Phase 2 | Express, MongoDB, Mongoose, SQL basics, JWT, bcrypt, validation |
| Phase 3 | Jest or Vitest, Supertest, Redis, WebSockets, GraphQL, webhook tooling |
| Phase 4 | Docker, Nginx, PM2, GitHub Actions, deployment infrastructure |

## Recommended Student Toolkit

- VS Code or equivalent editor
- Node.js LTS runtime
- Git and GitHub account
- Docker environment
- API client such as Postman or Bruno
- database GUI or shell tooling for MongoDB and SQL
